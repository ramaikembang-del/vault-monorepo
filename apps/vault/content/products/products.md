## Product Folder Structure

Each product follows this standardized structure:

```
products/[01-18]-[product-name]/
|-- README.md - Product overview
|-- product-strategy/ - Positioning, customers, roadmap
|-- specifications/ - PRD, feature specs, UI/UX
|-- technical/ - Architecture, database, deployment
|-- operations/ - Daily ops, onboarding, support
|-- integrations/ - POS systems, data sources
|-- marketing/ - Positioning, messaging, collateral
|-- sales/ - Pricing, demo scripts, tools
```
