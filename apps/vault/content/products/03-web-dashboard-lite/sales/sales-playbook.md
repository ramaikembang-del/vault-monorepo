# E1: Sales Playbook (Freemium Conversion)
**Product:** Web Dashboard Lite
**Model:** Freemium → Paid (Rp 0 → Rp 200-350K/month)
**Target:** Phase 1 Mikro owners (10-30 staff, Rp 80-250M revenue)

---

## Freemium Sales Motion (Different from A1)

| Aspect | A1 (Power BI Template) | E1 (Web Dashboard Lite) |
|--------|----------------------|------------------------|
| **Sales motion** | One-time close (Rp 2-15M) | Conversion funnel (Free → Paid) |
| **First touchpoint** | Sales call | Free signup (no human) |
| **Value demo** | Mockup/template preview | 7-day free usage |
| **Close trigger** | Pain + budget discussion | Feature limit + value realization |
| **Sales cycle** | 1-2 weeks (active selling) | 7-30 days (passive nurturing) |

**Key insight:** E1 sales is CONVERSION OPTIMIZATION, not traditional selling.

---

## The Freemium Funnel (E1 Journey)

```
Awareness → Free Signup → Activation → Engagement → Conversion → Retention
 ↓ ↓ ↓ ↓ ↓ ↓
 Landing Install + Use 3+ Use 7+ Hit limit Pay Rp
 page onboard features days or want 200-350K
 premium
```

**Our role:** Optimize each step, especially Activation → Engagement → Conversion.

## Phase 1: Free User Acquisition (Weeks 1-4)

### Goal
Get 100+ free signups in first month.

### Channels

**Primary: Facebook Groups**
- Post 2-3x/week in F&B owner groups
- Offer: "Dashboard gratis untuk track omzet & menu terlaris"
- CTA: Landing page link

**Secondary: WhatsApp Word-of-Mouth**
- Ask early users to share with other resto owners
- Referral incentive: "Ajak 3 teman, dapat premium 1 bulan gratis"

**Tertiary: Instagram Stories**
- Show dashboard screenshots
- "Gratis selamanya, no credit card"
- Link in bio

### Success Metric
- **Target:** 100 free signups in Month 1
- **Minimum:** 50 signups

---

## Phase 2: Activation (First 24 Hours)

### Goal
Get new user to complete onboarding and view first dashboard.

### Activation Checklist

 **User opens app**
 **Completes 2min tutorial**
 **Inputs first data point** (yesterday's revenue OR connects POS)
 **Views first metric** (Revenue graph OR Top 5 Menu)

### Onboarding Flow

**Step 1: Welcome Screen (30 sec)**
```
"Selamat datang di Dashboard Lite!

Dalam 2 menit, Anda akan lihat:
 Omzet harian
 Menu terlaris
 Performa staff

Mulai →"
```

**Step 2: Quick Input (60 sec)**
```
"Mari mulai dengan data kemarin:

Total omzet kemarin: Rp ______
Jumlah transaksi: _______

(Atau: Scan QRIS untuk import otomatis dari POS)"
```

**Step 3: First Dashboard View (30 sec)**
```
"Ini dashboard Anda!

[Mock revenue graph shown]

Tap setiap metrik untuk detail lebih.
Lanjut →"
```

### WhatsApp Follow-Up (Automated)

**If activated:** "Selamat! Anda sudah aktif. Ada pertanyaan?"
**If NOT activated in 24h:** "Halo! Butuh bantuan install dashboard? Saya bisa video call 10 menit. Reply 'YA' kalau perlu."

### Success Metric
- **activation Target:** 60% complete onboarding within 24h
- **Minimum:** 40% activation

---

## Phase 3: Engagement (Days 2-7)

### Goal
User opens app 3+ times in first week.

### Engagement Triggers

**Day 2: Nudge to input today's data**
WhatsApp: "Selamat pagi! Jangan lupa input omzet kemarin biar grafik Anda lengkap."

**Day 3: Feature discovery**
WhatsApp: "Sudah coba fitur Top 5 Menu? Tap 'Menu' di dashboard."

**Day 5: Milestone celebration**
WhatsApp: " Anda sudah track 5 hari! Lihat trend omzet 7 hari terakhir di dashboard."

**Day 7: Engagement check**
WhatsApp: "Sudah 1 minggu! Apakah dashboard ini membantu? Ada fitur yang kurang?"

### Success Metric
- **Target:** 50% users open app 3+ times in Week 1
- **Minimum:** 30% engagement

---

## Phase 4: Conversion (Days 8-30)

### Goal
Convert engaged free users to paid tier (Rp 200-350K/month).

### Conversion Triggers

#### Trigger 1: Feature Limit (Hard Gate)

**Free tier limits:**
- Max 6 metrics (no custom metrics)
- 30-day data history (no year-over-year)
- 1 user login
- No export PDF

**Paid unlock:**
- "Mau lihat 12 metrik lagi? Upgrade ke Premium." → CTA button

#### Trigger 2: Value Realization (Soft Nudge)

**After 7 days of usage:**
WhatsApp: "Mas/Mba [Name], sudah 1 minggu pakai dashboard. Apakah sudah terbantu?

Kalau mau fitur premium seperti:
 Custom metrics (buat KPI sendiri)
 Multi-user (kasih akses ke manager)
 Export PDF (buat laporan owner)

Harganya Rp 250K/bulan. Mau coba free trial 14 hari?"

#### Trigger 3: Pain Point Reminder

**After user views "blocked" metric 3+ times:**
In-app message: "Metric ini hanya untuk Premium. Upgrade sekarang, gratis 14 hari."

### Conversion Script (WhatsApp or In-App)

```
"Halo! Saya lihat Anda sudah aktif 2 minggu pakai Dashboard Lite.

Apakah fitur gratis sudah cukup, atau Anda butuh:
- Export PDF untuk laporan ke owner/investor?
- Lihat data 1 tahun penuh (bukan cuma 30 hari)?
- Custom metric (misalnya: Profit margin per menu)?

Kalau perlu, ada paket Premium Rp 250K/bulan.

Mau coba gratis 14 hari dulu? (No credit card)"
```

### Success Metric
- **Target:** 5-10% free users convert to paid in Month 1
- **Minimum:** 3-5% conversion

---

## Phase 5: Retention (Month 2+)

### Goal
Keep paid users subscribed (reduce churn).

### Retention Playbook

**Month 1: Onboarding + training**
- Weekly check-ins via WhatsApp
- "Feature of the week" tips
- Custom metric setup support

**Month 2-3: Engagement monitoring**
- If DAU drops <3x/week → proactive outreach
- "Sudah lama tidak lihat dashboard, ada masalah?"

**Month 6: Renewal prep**
- Show value report: "Dalam 6 bulan, Anda sudah track Rp XXM omzet!"
- Discount offer: "Perpanjang 6 bulan, diskon 15%"

**Churn prevention:**
- If payment fails → WhatsApp reminder (not just email)
- If user stops logging in → Call within 48h

### Success Metric
- **Target:** <10% monthly churn
- **Minimum:** <15% churn

## Pricing Tiers (E1 Specific)

| Tier | Price | Features | Target User |
|------|-------|----------|-------------|
| **Free** | Rp 0 | 6 metrics, 30-day data, 1 user | Testing, low engagement |
| **Premium** | Rp 250K/mo | 12 metrics, 1-year data, 3 users, PDF export | Single outlet, wants more depth |
| **Pro** | Rp 350K/mo | Unlimited metrics, unlimited data, 5 users, API access | 2-3 outlets, power user |

**Upsell path:** Free → Premium (most common) → Pro (rare, multi-outlet only)

## When to Involve Human Sales (vs Automated)

### Automated (Default)
- Free signup → onboarding
- Feature discovery nudges
- Standard upgrade offers

### Human Outreach (High-Value Signals)
- User views premium features 5+ times → Call
- User asks "How to X?" where X is premium → Offer demo
- User has >Rp 50M/month revenue (check during signup) → White-glove onboarding

**Rule of thumb:** If user seems like future Pro subscriber (multi-outlet), assign human touchpoint.
