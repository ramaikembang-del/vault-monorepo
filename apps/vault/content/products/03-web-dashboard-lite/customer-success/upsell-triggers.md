# E1: Upsell Triggers (Free → Premium/Pro)
**Purpose:** Identify optimal moments to offer paid upgrades
**Target Conversion:** 5-10% Free → Paid in Month 2

---

## Upsell Philosophy

**Don't sell, trigger natural conversion:**
- Users upgrade when they hit limits (hard gate)
- Users upgrade when they realize value (soft nudge)
- Users NEVER upgrade from spam (avoid over-messaging)

**Rule:** Offer upgrade when user demonstrates NEED, not want.

---

## Hard Triggers (Feature Gates)

### Trigger 1: Metric Limit (6 → 7th metric)

**When:** User tries to add 7th custom metric

**In-app message:**
```
Anda sudah pakai 6 metrik gratis.

Mau tambah metrik custom? (misal: Profit Margin, Waste %)

Upgrade ke Premium → 12 metrik total
Rp 250K/bulan. Free trial 14 hari.

[Upgrade Now] [Maybe Later]
```

**Conversion rate:** 15-20% (high intent—they actively tried to use blocked feature)

---

## Timing Rules

### DO offer upgrade:
- After 7+ days usage (proven engagement)
- When user hits feature limit (natural need)
- After value realization moment (positive sentiment)
- When health score >70 (engaged, likely to pay)

### DON'T offer upgrade:
- Day 1-3 (too early, no value proven)
- Health score <50 (not engaged)
- After negative feedback (fix problem first)
- More than once per week (spam)

---

## Upgrade Offer Sequence (Typical Journey)

**Day 8-14: First Soft Nudge**
- WhatsApp: "Sudah 1 minggu, mau coba Premium gratis?"
- If declined → Wait 2 weeks

**Day 15-30: Feature Gate Trigger as customer uses app
- User hits limit (7th metric, >30 day history, etc.)
- In-app offer with value prop
- If declined → Wait 2 weeks

**Day 30+: Value-Based Offer**
- High engagement + no conversion → Personal call
- "Kenapa belum upgrade? Ada yang kurang?"
- Custom offer based on feedback

## Objection Handling (During Upsell)

### Objection: "Gratis sudah cukup"

**Response:**
"Betul, gratis memang cukup untuk basic tracking.

Tapi kalau nanti butuh:
- Lihat data >30 hari (trend jangka panjang)
- Export PDF untuk laporan
- Kasih akses manager

Baru upgrade. Gak ada pressure sekarang. Keep using free!"

**Goal:** Plant seed, don't force

---

### Objection: "Terlalu mahal"

**Response:** (See `sales/pricing-justification.md` for full ROI breakdown)

"Boleh tanya, omzet per bulan berapa?

Kalau Rp 50-200M/bulan → Premium (Rp 250K) = 0.1-0.5% omzet.

ROI: Kalau save 5 jam/bulan (@ Rp 50K/jam) = Rp 250K → Break even.

Plus better decisions → Potensial hemat/earn Rp 1-2M/bulan.

Mau coba free 14 hari dulu? (No credit card)"

---

### Objection: "Nanti aja kalau udah butuh"

**Response:**
"Gak masalah! Free tier tetap ada.

Satu tip: Kalau nanti mau compare bulan ini vs 3 bulan lalu, harus upgrade ke Premium sekarang (biar data lama ke-save).

Kalau upgrade bulan depan, datanya cuma dari bulan depan onwards.

Just FYI!"

**Strategy:** FOMO - data loss if wait too long

## Premium vs Pro Decision Tree

**Recommend Premium (Rp 250K/mo) if:**
- Single outlet
- <5 staff needing access
- Needs custom metrics or PDF export
- Revenue Rp 50-200M/month

**Recommend Pro (Rp 350K/mo) if:**
- 2-5 outlets (multi-location)
- >5 users
- Needs API access
- Revenue >Rp 200M/month

**Decision script:**
```
Berapa outlet Anda?
→ 1 outlet: "Premium cocok untuk Anda"
→ 2-5 outlets: "Pro lebih worth it (per-outlet dashboard)"

Berapa orang perlu akses dashboard?
→ 1-3: "Premium (3 users)"
→ 4-5+: "Pro (5 users)"
```

## Trial Strategy

### 14-Day Free Trial (No Credit Card)

**Why no credit card:**
- Lower barrier (Indonesian market prefers no upfront commitment)
- Trust-building (not a trap)
- Higher trial signup rate

**Trial Period Plan:**

**Day 1-7: Feature Discovery**
- Email/WhatsApp: "Try custom metrics!"
- In-app tooltips for premium features
- Goal: Use 3+ premium features

**Day 8-10: Value Reinforcement**
- WhatsApp: "Sudah coba export PDF? Lihat hasilnya!"
- Show ROI: "Anda sudah save X hours dengan auto-export"

**Day 11-13: Conversion Push**
- WhatsApp: "Trial berakhir 2 hari lagi. Mau lanjut?"
- Offer: Pay now, get 1 month free (16% discount)
- Soft reminder (not aggressive)

**Day 14: Trial End**
- Email + WhatsApp: "Trial berakhir hari ini. Downgrade ke Free atau Continue Premium?"
- [Continue Premium Rp 250K/mo] [Stay Free]

**Trial→Paid Conversion Target:** 50-70%

## Red Flags (Don't Upsell)

**Avoid offering upgrade if:**
- Health score <50 (won't convert)
- User complained recently (fix issue first)
- Declined 2+ times already (respect decision)
- Using <3 days/week (not engaged enough)

**Better:** Let them churn gracefully (free tier), then win-back campaign later.
