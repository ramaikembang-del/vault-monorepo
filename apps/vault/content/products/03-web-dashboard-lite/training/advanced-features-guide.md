# E1 Dashboard Lite - Advanced Features Guide

**For:** Paid tier users or those considering upgrade

---

## Custom Date Ranges

**Beyond preset filters:**
1. Click date picker (top-right)
2. Select "Custom Range"
3. Enter start and end dates
4. Optional: Compare to previous period
5. Apply

**Saved filters:** Favorite your common ranges

---

## Team Management (Paid Tier)

**Add Team Members:**
1. Settings → Team
2. Click "Invite User"
3. Enter email
4. Select role:
 - **Admin:** Full access, can manage team
 - **Viewer:** Read-only access

**Use Cases:**
- Owner: Admin
- Managers: Viewer
- Accountant: Viewer (specific reports)

---

## Report Customization

**Available Reports (Paid):**
- Sales trends (daily/weekly/monthly)
- Item performance & rankings
- Category breakdown
- Payment methods analysis
- Hourly patterns
- Day-of-week analysis
- Customer transaction sizes
- Outlet comparisons (multi-location)
- ABC analysis (top products by revenue)
- Menu engineering matrix

**Filtering:**
- Multi-select outlets
- Filter by category
- Payment type
- Date ranges

---

## Scheduled Reports (Paid Tier)

**Auto-deliver reports to email:**

1. Reports → Select report
2. Click "Schedule" icon
3. Configure:
 - Frequency: Daily, Weekly, Monthly
 - Time: When to send
 - Recipients: Email addresses
 - Format: PDF or Excel

**Example:**
"Send daily sales summary to owner@restaurant.com at 8 AM"

---

## Data Export (Paid Tier)

**Export options:**
- Excel (.xlsx)

**What you can export:**

## Performance Tips

**If dashboard is slow:**
- Reduce date range
- Clear browser cache
- Use Wi-Fi (not cellular)
- Close other browser tabs

---

## Troubleshooting

**Data not updating:**
- Check POS sync status (Settings → Integrations)
- Manual refresh: Click sync icon
- Paid tier: Auto-syncs hourly

**Numbers different from POS:**
- Verify date range matches
- Check outlet filter
- Compare same metrics (revenue vs revenue, not revenue vs subtotal)
- Check if POS includes/excludes voided transactions

**Can't access reports:**
- Check your tier (Free = 3 reports, Paid = all)
- Upgrade to access more
