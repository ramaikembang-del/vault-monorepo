## Product-Specific Questions

**Q: What's the difference between Free and Paid tiers?**
A:
| Feature | Free | Paid (Rp 300K/mo) |
|---------|------|-------------------|
| Users | 1 | 5 |
| Data history | 30 days | Unlimited |
| Reports | 3 basic | 15+ advanced |
| Export | 5/month | Unlimited |
| Support | Email (48hr) | Email + WhatsApp (24hr) |
| POS sync | Manual daily | Auto hourly |

**Q: Can I try Paid tier for free?**
A: Use Free tier first. Upgrade anytime - no separate trial needed.

**Q: What happens to my data if I downgrade to Free?**
A: Data >30 days archived (not deleted). Re-activate by upgrading back.

**Q: How many outlets can I connect?**
A: Free: 1 outlet. Paid: Unlimited outlets.

**Q: Can I use E1 and A1 templates together?**
A: Yes, both can connect to same data source. Use whichever you prefer.

**Q: Is mobile app available?**
A: Mobile-responsive web app works on phones. Native app coming Q2 2025.

**Q: Can I white-label E1 for my clients?**
A: Not on E1. See C3 Advanced tier for white-labeling.

**Q: How do I add team members?**
A: Settings  ->  Users  ->  "Invite User" (paid tier only).

**Q: Dashboard not loading - blank screen**
A: Try:
1. Hard refresh (Ctrl+F5)
2. Clear browser cache
3. Try Chrome/Edge
4. Check [[biz/knowledge-base/troubleshooting/07-performance-issues|Performance Issues]]
