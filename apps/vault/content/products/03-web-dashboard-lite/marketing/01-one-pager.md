# E1: Marketing One-Pager
**Product:** Web Dashboard Lite (Freemium → Paid)
**Target:** Phase 1 Mikro resto owners (10-30 staff, Rp 80-250M/year)

---

## Headline
**"Dashboard Gratis untuk Restoran Anda—Lihat Omzet, Menu Terlaris, dan Performa Staff dalam 5 Menit"**

---

## Sub-Headline
Offline, aman, dan gampang. Tidak perlu kartu kredit. Data di HP Anda sendiri.

---

## Problem (Pain Points)

 **Manual Excel terlalu ribet** (10 jam/minggu cuma update angka)
 **POS report terbatas** (cuma angka mentah, gak ada analisis)
 **Gak tau menu mana yang untung vs rugi**
 **Susah track performa staff** (siapa yang produktif?)
 **Data hilang** (lupa backup Excel, HP rusak)

---

## Solution (Value Proposition)

 **Dashboard gratis selamanya** (6 metrik penting)
 **Offline-first** (data aman di HP, gak perlu internet)
 **Setup 5 menit** (install > input omzet kemarin > selesai)
 **Visual gampang dibaca** (grafik, bukan tabel Excel)
 **Support WhatsApp gratis** (bukan email)

---

## Features (Free Tier)

| Metrik | Kenapa Penting |
|--------|----------------|
| **Revenue harian** | Track omzet naik/turun setiap hari |
| **Top 5 menu** | Tau menu mana yang paling laku |
| **Staff performance** | Bandingkan sales per kasir/pelayan |
| **Customer count** | Berapa tamu hari ini vs kemarin |
| **Average transaction** | Omzet per tamu (naik berarti upsell berhasil) |
| **Busiest hours** | Jam berapa paling rame (buat scheduling staff) |

---

## How It Works (3 Steps)

**1. Install (2 menit)**
Download dari [link] seperti WhatsApp. Buka > daftar dengan email.

**2. Input data (30 detik/hari)**
Setiap pagi, input omzet kemarin. Atau connect POS untuk auto-sync.

**3. Lihat dashboard (kapan aja)**
Buka app, lihat grafik omzet 7 hari, menu terlaris, performa staff.

## FAQ (Quick)

**Q: Gratis selamanya atau trial aja?**
A: Gratis selamanya untuk 6 metrik dasar. Gak akan pernah kami cabut.

**Q: Datanya aman gak?**
A: 100% aman. Data di HP Anda, offline. Gak di server kami.

**Q: Ribet gak install?**
A: 5 menit. Seperti install WhatsApp. Ada video tutorial.

**Q: Harus bayar setelah berapa lama?**
A: Tidak pernah bayar kalau pakai Free. Bayar cuma kalau upgrade ke Premium (Rp 250K/bulan) untuk fitur tambahan.

**Q: Bisa multi-outlet?**
A: Free untuk 1 outlet. Pro (Rp 350K/bulan) untuk 2-5 outlet.
