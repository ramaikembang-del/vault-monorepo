# SaaS Platform Basic (C1)

**Product ID:** C1
**Sprint:** 3
**Pricing:** Rp 300-500K/month
**Target:** Phase 2 Mikro -> Kecil (1 location)

## Product Overview
The Basic tier of the PWBI F&B Growth SaaS platform. This cloud-based solution is designed for single-location operators ready to upgrade from the Lite app.

## Key Features
- **Cloud Sync:** Multi-device access
- **Advanced Reporting:** 10+ standard reports
- **User Management:** Single role (Owner)
- **Data Retention:** 1 year history
