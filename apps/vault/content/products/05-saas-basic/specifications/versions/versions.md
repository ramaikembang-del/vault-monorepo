## Version Naming Convention

**Format:** `[STAGE]_[MAJOR].[MINOR]`

**Stages:**
- `BETA` - Pre-launch testing (invite-only)
- `PUBLIC` - Production release (generally available)

**Versioning:**
- **Major:** Breaking changes, significant new features
- **Minor:** Bug fixes, small improvements
