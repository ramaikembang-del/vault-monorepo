# C1 Full SaaS - User Guide (Manager)

**Audience:** Location managers, operations managers
**Last Updated:** December 2025

---

## Your Role

As a manager, you:
- Monitor location performance daily
- Manage inventory & procurement
- Oversee staff scheduling
- Execute operational improvements

---

## Daily Workflow

**Morning (15 min):**
1. Review yesterday's sales & costs
2. Check inventory levels (reorder if needed)
3. Confirm today's staff schedule
4. Address any alerts

**Throughout Day:**
- Real-time sales monitoring
- Adjust operations as needed
- Log incidents/issues

**End of Day (10 min):**
- Close registers & reconcile
- Submit daily report
- Plan tomorrow

---

## Key Features for Managers

### 1. Location Dashboard
- Today's sales vs. target
- Product performance
- Staff performance
- Inventory status

### 2. Inventory Management
- Current stock levels
- Reorder recommendations
- Receive shipments
- Report waste

### 3. Staff Scheduling
- View schedule
- Request time-off approvals
- Track attendance
- Review performance

### 4. Sales & COGS Analysis
- Best/worst selling items
- Recipe costs vs. actuals
- Identify waste sources

---

## Common Tasks

**Task: Reorder Inventory**
1. Check **Inventory → Low Stock**
2. Review reorder recommendations
3. Create Purchase Order
4. Submit for approval (if > threshold)

**Task: Approve Staff Time-Off**
1. Navigate to **Staff → Requests**
2. Review request
3. Check schedule coverage
4. Approve or Deny

**Task: Report Broken Equipment**
1. **Operations → Incidents**
2. Create new incident
3. Upload photo
4. Assign to maintenance team
