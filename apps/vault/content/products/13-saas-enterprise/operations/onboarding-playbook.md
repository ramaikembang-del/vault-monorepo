# Enterprise Onboarding Playbook
**Product 13: SaaS Enterprise - White-Glove Onboarding**

**Owner:** Customer Success + Product  
**Status:** PLANNED - Year 3

---

## Enterprise vs SME Onboarding

| Aspect | SaaS Basic (SME) | SaaS Enterprise |
|--------|-----------------|----------------|
| **Timeline** | 7 days self-serve | 30-60 days hands-on |
| **Touch** | Automated emails | Dedicated CSM |
| **Customization** | None (standard) | High (bespoke features) |
| **Training** | Self-paced videos | On-site workshops |
| **Stakeholders** | 1-2 users | 5-20 users (IT, Finance, Ops) |

### Week 1: Kickoff & Discovery

**Kickoff Meeting** (90 min, all stakeholders)
- Introductions (Clario team + customer team)
- Project timeline & milestones
- Roles & responsibilities (who does what)
- Success criteria (what does "go-live" look like?)

**Discovery Workshop** (2-3 hours, on-site if possible)
- Business process mapping (how do they operate today?)
- Data sources audit (POS, accounting, inventory systems)
- Custom requirements (what features do they need?)
- Integration dependencies (must connect to ERP, etc.)

**Deliverables:**
- Project plan document (timeline, milestones, owners)
- Technical spec (integrations, custom features needed)

---

### Week 2-3: Data Migration & Integration

**Data Import:**
- Historical data (12-24 months transactions)
- User accounts (create logins for all stakeholders)
- Permissions setup (role-based access control)

**Integrations:**
- POS API connection (real-time sync)
- Accounting software link (Accurate, SAP)
- Custom data sources (if applicable - CRM, inventory)

**Testing:**
- Data validation (numbers match source systems?)
- Integration testing (real-time sync works?)
- UAT signoff (customer confirms accuracy)

---

### Week 4-5: Customization & Configuration

**Custom Features** (if contracted):
- Bespoke dashboards (C-suite view, ops view, finance view)
- Custom metrics (unique to their business model)
- White-label branding (their logo, colors)

**Configuration:**
- Alert rules (notify if revenue drops >X%)
- Report scheduling (weekly auto-emails)
- Benchmark setup (compare across their outlets)

---

### Week 6-7: Training

**Admin Training** (4 hours, on-site)
- **Audience:** IT + Finance leads
- **Topics:** User management, data management, troubleshooting
- **Format:** Hands-on workshop

**User Training** (2 hours, can be remote)
- **Audience:** Managers, analysts (5-20 people)
- **Topics:** Dashboard navigation, report generation, insights interpretation
- **Format:** Webinar + Q&A

**Executive Briefing** (30 min)
- **Audience:** C-suite (CEO, CFO, COO)
- **Topics:** High-level overview, ROI, strategic insights
- **Format:** Presentation deck

---

### Week 8: Go-Live

**Soft Launch** (Week 8)
- Enable for pilot group (2-3 locations, 5-10 users)
- Monitor usage, collect feedback
- Fix any issues

**Full Launch** (Week 10+)
- Enable for all locations, all users
- Announcement email (from their leadership)
- Cutover from old system (if replacing existing tool)

---

### Week 9-12: Stabilization & Optimization

**Daily Check-Ins** (Week 9-10)
- CSM reaches out daily (Slack/email)
- Address any issues immediately
- Monitor usage stats

**First Month Review** (Week 12)
- Usage report (who's using, how often?)
- Satisfaction survey (NPS)
- Next-phase planning (additional features, training)

---

## Enterprise CSM Responsibilities

**Pre-Launch (Weeks 1-8):**
- Project management (timeline, deliverables)
- Stakeholder coordination (many people to align)
- Escalation point (internal + external issues)

**Post-Launch (Ongoing):**
- **Quarterly Business Reviews (QBRs):** Strategic check-ins
- **Monthly Usage Reports:** Data-driven insights
- **Ad-Hoc Support:** Priority access (Slack channel, phone)
- **Feature Requests:** Advocate for customer needs to Product team

---

## Enterprise Success Metrics

**Onboarding Phase:**
- On-time delivery (<10 weeks)
- UAT pass rate (>95% data accuracy)
- Training completion (100% of trained_users certified)

**Post-Launch:**
- Adoption rate (>80% of users active monthly)
- NPS (>8 for enterprise)
- Renewal rate (>95% for enterprise)

---

## Escalation Path

**Standard Issues:** CSM handles  
**Technical/Complex:** Escalate to Product team  
**Executive Concerns:** Escalate to Founder

**Enterprise SLA:** <2 hour response, <8 hour resolution (P0/P1)
