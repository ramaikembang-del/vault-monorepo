## CUSTOMIZATION GUIDE

### Branding
**Logo:** SVG/PNG, 200×50px, uploaded via Settings
**Colors:** Primary, secondary, accent (hex codes)
**Domain:** analytics.yourcompany.com (CNAME: c4.pwbi.co)
**Remove PWBI:** +Rp 500M/year (white-label package)

### Dashboards
**Custom Widgets:** Revenue/sqft, table turnover rate, etc.
**Builder:** Drag-drop interface
**Saved Views:** Per user, role, brand
**Cost:** Rp 5M per custom dashboard

### Workflows
**Approval Chains:** Regional Mgr → Country Dir → CFO
**Custom Fields:** Capture unique data (halal cert expiry, etc.)
**Business Rules:** Alert if waste > 5%, labor > 30%
**Cost:** Quoted (complexity-based)

### Reports
**Templates:** Modify or create new
**Scheduled:** Email/Slack daily/weekly
**Branding:** Your logo on all reports
**Cost:** Minor changes included, complex quoted

## SALES PLAYBOOK

### Target
**Profile:** 100+ locs, Rp 100B+ revenue, manual reporting pain
**Committee:** CTO (tech), CFO (budget), COO (ops), CEO (final)

### Process (6-12 months)
1. Discovery (pain, goals)
2. Demo (customized)
3. PoC (5-10 locs, 2-4 weeks)
4. Proposal (pricing, ROI)
5. Security review (CISO)
6. Negotiation (price, terms)
7. Close

### Objections
**"Too expensive":** ROI calc (10x return)
**"Too complex":** Phased rollout
**"Migration risk":** Insurance, parallel run

### Win Strategies
**Champion:** Arm CFO with data
**Urgency:** Audit deadline, expansion
**Proof:** PoC, reference visit
