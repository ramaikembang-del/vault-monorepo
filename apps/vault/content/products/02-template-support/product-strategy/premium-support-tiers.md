# Premium Support Tiers (R2 & R3)
**Date:** December 31, 2025
**Purpose:** Define R2 Premium and R3 Enterprise support offerings
**Context:** Extends R1 Basic (existing) with higher-tier options

---

## SUPPORT TIER OVERVIEW

Product 02 (Template Support) offers three tiers:

| Tier | Price | Target | Key Feature |
|------|-------|--------|-------------|
| **R1 Basic** | Rp 75K/mo | All customers | Template updates + async support |
| **R2 Premium** | Rp 175K/mo | Tier 2/3 customers | R1 + monthly call + 1hr custom work |
| **R3 Enterprise** | Rp 500K/mo | High-touch customers | R2 + consulting + 2hr custom work |

**Design Philosophy:** TIME-BASED limits, not feature-based

---

## R2: PREMIUM SUPPORT

### Target Customer:
- Tier 2/3 template buyers
- Needs occasional customizations
- Wants monthly check-ins
- Values faster response

### Service Catalog:

**Everything in R1 Basic PLUS:**

**5. Monthly Check-In Call (30 min)**
- Review dashboard usage
- Answer strategic questions
- Identify improvements
- Conversational, no formal agenda

**6. Priority Support (<12hr response)**
- Faster than R1's 24hr
- WhatsApp preferred (not just email)
- Weekend emergencies (best-effort)

**7. Custom Work Allocation: 1 hour/month**
- Add/modify 1-2 metrics
- Adjust calculations
- Minor visual tweaks
- **STRICT TIME CAP:** Timer tracked

### Service Level Agreement:

| Metric | Commitment |
|--------|------------|
| **Response Time** | <12 hours (priority queue) |
| **Monthly Call** | 30 min (scheduled) |
| **Custom Work** | 1 hour/month (hard cap!) |
| **Template Updates** | Monthly + subscriber requests |
| **Support Channel** | WhatsApp priority + email |
| **Rollover** | No - use it or lose it |

### Custom Work Examples (1 hour scope):

 **Fits in 1 hour:**
- Add 2-3 simple metrics
- Modify existing calculation
- Rearrange dashboard page
- Update color scheme

 **Exceeds 1 hour:**
- Build entirely new page (90-120 min)
- Complex custom logic (>1hr)
- Multiple interconnected metrics

**Overage:** Rp 250K/hour for additional time

### Operational Model:

**Time Budget:** 3-4 hours/month per customer
- 0.5 hr: Faster async support
- 0.5 hr: Monthly check-in call
- 1 hr: Custom work (if used)
- 0.5 hr: Template updates
- 0.5-1 hr: Admin, coordination

**Scale:** Can handle 15-20 customers solo

### Unit Economics:

| Item | Value |
|------|-------|
| **Price** | Rp 175K/month |
| **Variable Cost** | Rp 70K (4hr @ Rp 175K/hr ÷ 10 avg) |
| **Gross Margin** | 60% (Rp 105K) |
| **Breakeven** | 6 subscribers (covers fixed costs) |

**Verdict:** Profitable if time limits enforced

### Target Customer:
- Tier 3 template buyers (5+ outlets)

### Service Catalog:

**Everything in R2 Premium PLUS:**

**8. Strategic Consulting (60 min/month)**
- Monthly business review call

| Metric | Commitment |
|--------|------------|

| **Monthly Call** | 60 min strategic consulting |

| **Monitoring** | Weekly proactive checks |
| **Support Channel** | Dedicated WhatsApp + phone |
| **Rollover** | No - use it or lose it |

### Custom Work Examples (2 hours scope):

 **Fits in 2 hours:**
- Build new dashboard page
- Add 5-7 related metrics
- Complex calculation formulas
- Integration with new data source

 **Exceeds 2 hours:**
- Rebuild entire dashboard (>>2 hours)

**Overage:** Rp 300K/hour for additional time

**Time Budget:** 5-6 hours/month per customer
- 1 hr: Strategic consulting call
- 0.3 hr: Quarterly review (amortized)
- 2 hr: Custom work (if used)
- 0.5 hr: Proactive monitoring
- 0.5 hr: Priority support
- 0.5-1 hr: Templates + admin

**Scale:** Can handle 8-10 customers solo (boutique service)

### Unit Economics:

| Item | Value |
|------|-------|
| **Price** | Rp 500K/month |
| **Variable Cost** | Rp 112.5K (6hr @ Rp 150K/hr ÷ 8 avg) |

| **Breakeven** | 2 subscribers |

**Verdict:** Highly profitable (if time limits enforced)

## CRITICAL SUCCESS FACTORS

### 1. STRICT TIME ENFORCEMENT

**Problem with old model:**
- "2 customizations/month" → Vague, leads to scope creep
- Customer: "Rebuild my entire sales page = 1 customization?"

**New model:**
- R2: 1 hour/month (hard stop)
- R3: 2 hours/month (hard stop)
- Customer sees: "0.5 hours used this month (0.5 remaining)"

**No Rollover:** Unused hours don't carry forward

### 2. PRE-DEFINED SERVICE CATALOG

**Create time estimates:**
- "Add simple metric" = 15 min
- "Modify calculation" = 20 min
- "Build new page" = 90-120 min
- "Reorganize layout" = 30 min

**Customer self-selects:**
> "I have 1 hour. I'll do 2 simple metrics + 1 layout change."

### 3. TRANSPARENT OVERAGE PRICING

**If customer needs >2 hours:**
- R2: Rp 250K/hour additional
- R3: Rp 300K/hour additional
- OR upgrade to next tier

**This protects margins!**

## IDEAL CUSTOMER SCENARIOS

### When to Choose R1 Basic:
- "I just want template to stay updated"
- "I'm self-sufficient, don't need custom work"
- "Budget-conscious"
- **Persona:** Tech-savvy Tier 1 buyer

### When to Choose R2 Premium:
- "I need occasional tweaks, 1-2 times/month"
- "I want monthly check-ins for accountability"
- "Faster support matters to me"
- **Persona:** Growing Tier 2/3 operator (3-4 outlets)

### When to Choose R3 Enterprise:
- "I need ongoing strategic partner"
- "My time is worth more than DIY"
- "I want proactive monitoring, not reactive fixes"
- **Persona:** Delegating Tier 3 buyer (5+ outlets)

## LAUNCH STRATEGY

### Phase 1: Launch R1 Only (Sprint 1)
- Get basic support working
- Validate 30% conversion
- Learn support patterns

### Phase 2: Add R2 Premium (Month 3-4)
- After 10+ R1 customers
- Offer upgrade path
- Test 1-hour/month model

### Phase 3: Add R3 Enterprise (Month 6)
- After R2 validated
- Boutique service (max 10 customers)
- Strategic partnerships

**Why phased:** Don't build R2/R3 until R1 proves demand!
