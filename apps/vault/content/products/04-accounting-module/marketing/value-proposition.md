# Value Proposition - R2 Accounting Module
**Product:** R2 - Analytics Suite / Accounting Module
**Target:** Phase 1-2 Mikro F&B (1-3 outlets)
**Core Promise:** "P&L in 5 Minutes, Not 5 Hours"

---

## Value Proposition Canvas

### Customer Segment: Mikro F&B Owners

**Jobs to Be Done:**
1. Track monthly profit/loss (know if business is profitable)
2. Manage cash flow (avoid running out of money)
3. Prepare financial statements for bank loans
4. Make data-driven decisions (menu pricing, cost cutting)

**Pains:**
1. Excel accounting takes 5+ hours/month (time-consuming)
2. Manual errors in calculations (unreliable)
3. Not SAK EMKM compliant (bank rejects Excel P&L)
4. Can't afford full accountant (Rp 3-5M/month too expensive)
5. Existing software too complex (Jurnal/Accurate for enterprises)

**Gains:**
1. Save time on accounting (spend on business growth instead)
2. Accurate financial picture (make better decisions)
3. Bank-ready statements (get loans approved)
4. Affordable solution (within Mikro budget)
5. Simple to use (no accounting expertise needed)

**2. 98% Cheaper Than Accountant**

- Monthly P&L follows Indonesian Mikro accounting standard
- Bank accepts for KUR loan applications (Rp 50-500M loans)
- **Value:** Unlock Rp 50-500M financing for expansion

**4. F&B-Specific (Not Generic)**
- 10 expense categories tailored for F&B (COGS, labor, delivery, etc.)
- Not 50+ generic categories (confusing for Mikro owners)
- **Benefit:** Faster data entry, clearer insights

**5. Simple UX (No Training Needed)**
- WhatsApp expense entry (no app login required)
- 15-minute setup (not 2-4 hours like Jurnal)
- **Benefit:** Start using Day 1, no learning curve
