## Overview

Comprehensive Power BI dashboard providing daily F&B performance analytics with flexible 3-tier pricing:

### **Tier 1: DIY Template** (Rp 3.5-6M one-time, modular)
- Pre-built .pbix file + documentation
- 30-day email support
- **You set it up yourself**
- Target: Tech-savvy, budget-conscious

### **Tier 2: Assisted Setup** (Rp 5-7M one-time)
- Everything in Tier 1
- **+ 2-3 hours remote setup assistance**
- 90-day email support + 1 free update
- Target: Needs implementation help

### **Tier 3: Fully Managed** (Rp 10-15M + Rp 300-500K/month)
- Everything in Tier 2
- **+ Full setup + ongoing support & maintenance**
- Monthly maintenance, quarterly reviews, priority support
- Optional: Monthly strategic consulting
- Target: "Set it and forget it" + strategic partner
