## Pricing Psychology

### Why Modular Pricing Works

1. **Customer Choice** - Pay only for what you need
2. **Clear Value Ladder** - Each module = +Rp 1.25-1.5M (easy math)
3. **Prevents Feature Creep** - Want more? Buy the module.
4. **Natural Upsell Path** - Start small, grow over time

### Anchoring Strategy

**Always present in this order:**
1. Show T2 first (sweet spot, most customers)
2. Compare up (T3: more expensive, more service)
3. Compare down (T1: cheaper, DIY effort)

**Script:**
> "Most customers choose Professional (T2) at Rp 7-12M. We set it up together, saves you 6+ hours. If you're very technical and want to DIY, Essentials (T1) starts at Rp 3.5M. If you want fully managed ongoing support, Enterprise (T3) is Rp 10M+ setup plus optional support (see Product 02). Which sounds right for you?"
