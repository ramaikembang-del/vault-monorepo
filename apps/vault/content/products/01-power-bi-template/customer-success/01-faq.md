## Troubleshooting

### Q16: "Data source error" - how to fix?

**A

:** Excel file moved. Fix:
1. Power BI -> "Transform Data"
2. "Data Source Settings" -> "Change Source"
3. Browse to new Excel location

---

### Q17: Food cost shows 150% (impossible!)

**A:** Data entry error - revenue entered as thousands instead of full amount. Fix Excel amounts, refresh dashboard.

---

### Q18: Dashboard is blank

**A:** 3 causes:
1. No data entered (fill Excel first)
2. Wrong date format (use DD/MM/YYYY)
3. Can't find Excel files (see Q16)
