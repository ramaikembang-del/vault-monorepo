# A1 Power BI Template - Quick Start Guide (Complete Ecosystem)

**Duration:** 30 minutes
**Goal:** Get your first dashboard working with real data using the ecosystem bundle

---

## Step 0: Understand Your Ecosystem Bundle (2 minutes)

**You received 5 components:**

1. **Power BI Dashboard** (.pbix) - The visualization engine
2. **4 Excel Templates** - Pre-formatted data entry ( THE KEY!)
 - Daily-Sales.xlsx
 - Food-Cost.xlsx
 - Staff-Log.xlsx
 - Outlet-Summary.xlsx (Tier 2/3 Professional/Enterprise)
3. **Google Sheets** - Mobile-friendly alternatives
4. **Automation** - One-click import Excel -> Power BI
5. **Video Tutorials** - 10-minute learning path

**Recommended Workflow:**
1. Fill Daily-Sales.xlsx with today's sales (5 min)
2. Open Power BI Dashboard
3. Click "Refresh" button (30 seconds)
4. Dashboard updates with your data!

**30-minute time-to-insight achieved!**

---

## Step 1: Start with Excel Templates (Excel-First Approach)

**Why Excel first?**
- Excel is familiar (staff can help)
- Most time spent on data entry, not Power BI setup
- If Excel works, Power BI is just 5 minutes

**Quick Test:**
1. Open `Daily-Sales.xlsx`
2. Notice dropdowns (Product, Staff, Payment)
3. Fill ONE day of sample sales
4. Save file

**Tier 2/3 (Professional/Enterprise - 50-150 metrics):** Your templates are PRE-CUSTOMIZED to your POS format - just import CSV!

---

## Step 2: Import Your Data to Excel

### Method 1: Manual Entry (Tier 1 Essentials - 30-50 metrics - Learning)
1. Fill Daily-Sales.xlsx for 3-7 days
2. Use dropdown lists (error-free!)
3. Save file in known location

### Method 2: POS Export Import (Tier 2/3 Professional/Enterprise - 50-150 metrics - Production)
1. Export CSV from POS
2. Open Excel -> Data -> From Text/CSV
3. Select your POS export
4. Data fills template automatically!

---

## Step 3: Connect Power BI to Excel

1. Open Power BI Dashboard (`.pbix` file)
2. You'll see "sample" data
3. Home -> Get Data -> Excel
4. Select your Daily-Sales.xlsx
5. Choose tables to import
6. Click "Load"

**Power BI now connected to YOUR Excel!**

---

## Step 4: Refresh & Explore

1. Home -> **Refresh** button (wait 30 sec)
2. Dashboard updates with your data!
3. Navigate pages:
 - **Sales Overview:** Daily/weekly/monthly trends
 - **Item Performance:** Best sellers, profitability
 - **Staff Leaderboard:** Performance tracking
 - **P&L:** Revenue, costs, profit

## Common First-Time Issues

**"Excel won't import to Power BI"**
- Save Excel file, then refresh Power BI
- Date format must be YYYY-MM-DD
- No blank rows in Excel

**"Dropdown not working in Excel"**
- Click cell, use dropdown arrow (don't type)
- Re-select cell if needed

**"Google Sheets instead?"**
- Use make-a-copy links (included in bundle)
- Mobile-friendly for on-the-go entry
