## Case Study Template (Storytelling)

### "How Dimas Saved Rp 3M/Month Using Excel + Power BI"

**Structure:**

**Before:**
```
Dimas punya warung kopi di Yogya.
Revenue Rp 35 juta/bulan.

Problem:
- Excel berantakan (20+ sheets)
- Food cost 'kayaknya' 30% (gak tau pasti)
- Nggak tau produk mana yang rugi

Pusing liat angka doang. Susah ambil keputusan.
```

**The Trigger:**
```
Buka outlet ke-2. Sekarang ada 2 file Excel.

Mau compare performa Outlet A vs B?
-> Copy-paste manual. 2 jam.
-> Sering salah rumus. Frustasi.

"Gue butuh system."
```

**The Solution:**
```
Ketemu Power BI Template mulai Rp 3.5 juta (modular).

Awalnya skeptical: "Gue udah punya Excel, kenapa harus bayar Rp 3-4 juta?"

Tapi liat demo 2 menit -> "Wah, ini exactly yang gue butuh." Pilih T1-Standard (Rp 4.5M).

Setup 3 jam (weekend project).
- Import 2 file Excel (Outlet A, Outlet B)
- Dashboard langsung jadi
```

**The Aha Moment:**
```
Week 1: Lihat dashboard pertama kali.

SHOCK.

Food cost ternyata 38%, bukan 30%.
= Bocor Rp 3 juta/bulan (8% x Rp 35 juta).

"Gue nggak pernah tau ini. Excel gue nggak kasih tau."
```

**The Result:**
```
Month 1:
- Ganti supplier sayur (lebih murah 15%)
- Kontrol portion size (dari 250g -> 220g)
- Food cost turun dari 38% -> 32%
- Hemat Rp 2 juta/bulan

Month 3:
- Dashboard udah balik modal (Rp 4.5 juta hemat sudah terkumpul)
- Buka outlet ke-3
- Upgrade ke Tier 2 (Rp 8 juta) buat multi-location + P1 module

Quote:
"Power BI ini kayak kacamata. Data gue ada di Excel, cuma gue nggak bisa 'lihat' dengan jelas. Sekarang bisa."
```

## Key Takeaways: Making It Sexy

### 1. **Metaphor Matters**
- "Kacamata" = VISUAL, RELATABLE, NON-THREATENING
- Not "upgrade your system" (scary)
- But "upgrade your vision" (empowering)

### 2. **Show, Don't Tell**
- Demo video 2-3 minutes MAX
- Live import Excel -> Dashboard (30 seconds)
- Before/After side-by-side comparison

### 3. **Remove ALL Risk**
- 30-day money-back (no questions asked)
- "Try before you buy" mentality
- Free demo/sample dashboard

### 4. **Social Proof EVERYWHERE**
- 50+ customers (number)
- Real names (Dimas, Amanda, Budi)
- Specific results (Rp 3M/month saved, 8% food cost)

### 5. **Price Anchoring**
- NOT "Rp 3.5 juta" (sounds expensive)
- BUT "Rp 3.5 juta vs Rp 50 juta consultant" (sounds cheap)
- OR "Rp 3.5-6 juta sekali vs Rp 1 juta/bulan selamanya" (sounds smart)
- OR "Mulai dari Rp 3.5 juta (30 metrics) - modular, pilih yang kamu butuh" (sounds flexible)
