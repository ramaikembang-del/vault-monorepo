# A4d Supplier Manager - Complete Documentation

## Product Brief

**Product:** A4d - Supplier Manager
**Pricing:** Rp 500K/year
**Launch:** Q1 2025

### Problem
Managing multiple suppliers manually leads to:
- Missed price opportunities (5-10% savings possible)
- Poor delivery tracking (late deliveries disrupt operations)
- No supplier performance data (can't identify best/worst suppliers)
- Manual PO creation (time-consuming)

**Financial Impact:** Rp 5-10M/year in avoidable costs

### Solution
Centralized supplier management with performance tracking, price comparisons, and auto-notifications.

### Key Features
1. **Supplier Database:** Store contacts, products, prices, terms
2. **Performance Tracking:** On-time delivery %, quality scores, price consistency
3. **Price Comparison:** Compare prices across suppliers for same item
4. **Auto-Reorder:** Integration with A4a (inventory alerts  ->  auto-create PO)
5. **Invoice Matching:** 3-way matching (PO  ->  Delivery  ->  Invoice)

### Value
- Better prices: 5-10% savings = Rp 5-10M/year
- Reduce stock-outs: 80% fewer (reliable suppliers)
- Time saved: 3 hours/week procurement

**ROI:** 10-20x

## User Guide

### Setup (15 min)
1. Add suppliers (name, contact, products they supply, lead times)
2. Import price lists (Excel or manual)
3. Connect to A4a (inventory integration)

### Daily Use (10 min)
- Review reorder alerts  ->  Click "Create PO"
- Send PO to supplier (email/WhatsApp)
- Mark as sent
- Track delivery status

### Supplier Performance
- Dashboard shows:
 - On-time delivery rate
 - Price stability
 - Quality scores (rate each delivery 1-5 stars)
 - Response time

**Quarterly:** Review worst performers  ->  renegotiate or switch suppliers

## FAQ

**Q: Can I track multiple suppliers per item?**
A: Yes, compare who has best price/delivery/quality.

**Q: Does it send Purchase Orders automatically?**
A: PO created automatically (from A4a reorder alerts), you review & send.

**Q: What if supplier doesn't have email?**
A: Send via WhatsApp or print PDF.
