# A4d - Technical Specs | Pricing | FAQ

## Technical Specs

**Platform:** Mobile + Web
**Tech:** React, Node.js, PostgreSQL

**Core Features:**
- Supplier CRM database
- Price comparison engine
- PO generator (PDF templates)
- Delivery tracking workflow
- Invoice 3-way matching
- Performance analytics

**Integrations:**
- A4a (reorder alerts → auto PO)
- Accounting (Accurate, Jurnal)
- WhatsApp Business API

## FAQ

**Q: How many suppliers can I add?**
A: Unlimited.

**Q: Can I track deliveries?**
A: Yes, PO status tracking (sent → in-transit → delivered).

**Q: Integration with accounting?**
A: Yes, export to Accurate, Jurnal, QuickBooks.

**Q: What if supplier price increases?**
A: Alert sent, can compare to other suppliers.
