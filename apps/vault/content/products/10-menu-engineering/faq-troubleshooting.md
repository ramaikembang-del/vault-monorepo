# A4b Menu Engineer - FAQ

---

## General

**Q: What is A4b Menu Engineer?**
A: Tool that calculates true menu item profitability and recommends pricing/menu changes to maximize profit.

**Q: How much does it cost?**
A: Rp 500K/year. 14-day free trial.

**Q: Do I need a POS?**
A: No, but recommended for sales volume data. Can use standalone.

---

## Features

**Q: How accurate are cost calculations?**
A: Very accurate if you input correct ingredient costs & portions. Update costs monthly for best results.

**Q: Can it handle complex recipes?**
A: Yes, multi-level BOMs supported (e.g., sauce within main dish).

**Q: What about overhead (rent, utilities)?**

## Pricing Recommendations

**Q: Should I trust AI pricing recommendations?**
A: Use as guidance, not gospel. Consider your brand positioning, target market. Test price changes gradually.

**Q: How does it know market prices?**
A: Web scraping + user-submitted data + competitive intel.

## Troubleshooting

**Problem: Margin calculations seem wrong**
- Check: Ingredient costs current? Portion sizes accurate?
- Fix: Update costs, verify portions

**Problem: Can't find ingredient in database**
- Solution: Add custom ingredient
