# A4b Menu Engineer - User Guide

**Setup:** 20 minutes | **Daily use:** 10 minutes/week

---

## Quick Start

**Step 1: Import Menu (5 min)**
- Connect POS OR upload Excel
- Verify all items imported
- Categorize (drinks, food, desserts)

**Step 2: Add Recipe Details (10 min)**
For each item:
- List ingredients & quantities
- Set portion sizes
- Add labor time (avg minutes to prepare)

**Pro Tip:** Use bulk import template for faster setup

**Step 3: Set Costs (5 min)**
- Import supplier price list OR
- Enter ingredient costs manually
- Update monthly

---

## Key Features

### Profitability Dashboard

**View:** All items sorted by margin %

**Color Coding:**
- Red: < 40% margin (CONCERN)
- Yellow: 40-60% margin (OK)
- Green: > 60% margin (EXCELLENT)

**Actions:**
- Tap item → See full breakdown
- Filter by category
- Export report (Excel/PDF)

### Recipe Builder

**Add New Item:**
1. Item name: "Cappuccino Large"
2. Selling price: Rp 30,000
3. Ingredients:
 - Espresso (30g): Rp 3,000
 - Milk (200ml): Rp 4,000
 - Sugar (10g): Rp 100
 - Cup: Rp 500
4. Labor: 4 min @ Rp 25K/hour = Rp 1,667
5. Overhead: 10% = Rp 927

**Total COGS:** Rp 10,194
**Gross Profit:** Rp 19,806
**Margin:** 66%

### Pricing Recommendations

**Example Alert:**
```
 PRICING OPPORTUNITY

"Nasi Goreng Spesial"
Current: Rp 25,000 (margin: 42%)
Market average: Rp 30,000
Recommendation: Increase to Rp 28,000
Impact: Margin → 55%, sales volume -5% (acceptable)
Net profit increase: Rp 3M/year

[Accept] [Customize] [Reject]
```

### Menu Optimization

**4-Quadrant Analysis:**

| | High Volume | Low Volume |
|---|---|---|
| **High Margin** | STARS<br/>Promote heavily | GEMS<br/>Promote more |
| **Low Margin** | FIX IT<br/>Re-engineer cost | DOGS<br/>Discontinue |

**See what others charge:**
- Your price: Rp 35K

- Competitor B: Rp 32K
- Market avg: Rp 36K
- **Insight:** You're priced competitively

## Weekly Tasks

**Monday Morning (10 min):**
- Review profitability dashboard
- Check if ingredient costs changed
- Act on recommendations (price changes, removals)
- Update recipes if changed

**Monthly (30 min):**
- Update all ingredient costs (from suppliers)
- Physical count check (portions accurate?)
- Remove seasonal items
- Add new items

---

## Tips

 Update ingredient costs monthly (prices change!)
 Include ALL costs (don't forget packaging, condiments)
 Track portion sizes (kitchen staff consistency)
 Test price changes gradually (A/B test)
 Review competitor prices quarterly
