# Templates Hub

**Purpose:** The central library for all standardized documents, spreadsheets, and presentation decks used across PWBI F&B Growth execution. This directory ensures consistency and speed in our "Smart Solo" operations.

---

## Template Categories

### Internal Templates
**Location:** `/internal/`

Operational tools for running the company.
- **Project Management:** Sprint Planning, Roadmaps, OKRs.
- **Operations:** Meeting Agendas, Status Reports, Incident Response.
- **HR & Team:** Job Descriptions, Review Forms, 1-on-1s.
- **Product:** PRDs, Tech Specs, Release Notes.

See: [[biz/templates/internal/internal|Internal Templates README]]

### Product Documentation Templates
**Location:** `/product-docs/`

Standardized formats for external-facing product documentation.
- **User Guides:** Structure for "Getting Started" manuals.
- **API Reference:** Standard format for API docs.
- **Release Notes:** Public changelog format.

See: [[biz/templates/product-docs/product-docs|Product Docs Templates README]]

---

## Usage Policy

1. **Don't Edit the Master:** Always duplicate the template file before using it.
2. **Versioning:** If you improve a template, save it as `v2` and propose a merge to the master folder.
3. **Standardization:** Maintain the "PWBI Brand Voice" (Professional, Direct, Actionable) in all new templates.
