# Implementation Partner Training

**Last Updated:** 2025-12-27
**Duration:** 3 days (24 hours)
**Format:** Hands-on technical training

---

## Prerequisites

- Technical background (IT consultant, systems integrator)
- Basic understanding of data analytics and accounting
- Familiarity with F&B operations (helpful)

---

## Day 1: Technical Foundations (8 hours)

### Module 1: Product Architecture (3 hours)
- System architecture overview
- Data flows (POS → PWBI → Reports)
- Security model (authentication, RLS)
- API overview

**Read:** [[products/shared-technical/shared-technical|Architecture Docs]]

### Module 2: POS Integrations (3 hours)
- Supported POS systems (Pawoon, Moka, Qasir, Olsera)
- Integration methods (webhooks, API polling, CSV)
- Authentication (OAuth, API keys)
- Data mapping

**Hands-on:** Connect to test POS system

**Read:** POS Connectors documentation

### Module 3: Data Platform (2 hours)
- Database schema
- Data retention policies
- Row-level security
- Backup and recovery

---

## Day 2: Implementation Process (8 hours)

### Module 4: Customer Onboarding (2 hours)
- Discovery and scoping
- Technical requirements gathering
- Implementation planning
- Timeline and milestones

### Module 5: System Setup (4 hours)
**Step-by-step:**
1. Account creation and configuration
2. POS integration setup
3. Data migration (historical data)
4. User access and permissions
5. Custom configurations
6. Testing and validation

**Hands-on:** Complete full implementation for test customer

### Module 6: Training End Users (2 hours)
- Training customers (restaurant owners/managers)
- Creating custom training materials
- Best practices for change management

---

## Day 3: Advanced Topics & Troubleshooting (8 hours)

### Module 7: Customization (3 hours)
- Custom reports and dashboards
- Data transformation logic
- Advanced Power BI techniques (for A1)
- API usage (for C1+)

### Module 8: Troubleshooting (3 hours)
- Common issues and solutions
- Integration debugging
- Data accuracy validation
- Performance optimization

**Practice:** Troubleshoot common scenarios

### Module 9: Support & Escalation (2 hours)
- Support SLAs for partners
- Escalation procedures
- Partner portal tools
- Accessing PWBI support team

## Sample Implementation Timeline

**Small Customer (1 outlet, 1 POS):** 1-2 weeks
- Week 1: Setup and integrate
- Week 2: Training and go-live

**Medium Customer (3-5 outlets, multiple POS):** 3-4 weeks
- Week 1: Discovery and planning
- Week 2-3: Implementation
- Week 4: Training and go- live

**Enterprise (10+ outlets):** 6-12 weeks
- Phased rollout recommended

---

## Success Metrics

**Good Implementation Partner:**
- Average implementation time meets benchmarks
- <5% re-work rate
- >85% customer satisfaction (post-implementation survey)
- <10% churn in first 3 months (customers you implemented)
