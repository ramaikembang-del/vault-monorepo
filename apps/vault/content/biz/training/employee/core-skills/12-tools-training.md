# Tools Training

**Last Updated:** 2025-12-27
**For:** All employees

---

## Essential Tools (Everyone)

### 1. Slack - Team Communication

**Setup:**
- Download desktop + mobile apps
- Join channels: #general, #your-team, #random
- Set notification preferences
- Add profile photo and bio

**Best Practices:**
- Use threads for conversations
- @mention for urgent
- Use channels, not DMs (transparency)
- Set status when away/busy

**Key Channels:**
- #general: Company announcements
- #engineering, #product, #sales, #customer-success: Team channels
- #questions: Ask anything!
- #wins: Celebrate victories
- #random: Fun stuff

---

### 2. Google Workspace - Email & Docs

**Gmail:**
- Use your@pwbi.id email for all work communication
- Enable 2FA
- Set up email signature

**Google Drive:**
- All company files in shared drives
- Don't store sensitive data in personal Drive
- Request access if needed

**Google Docs/Sheets:**
- Collaborate in real-time
- Use comments for feedback
- Share with specific people, not "anyone with link"

---

### 3. Notion - Project Management & Wiki

**What We Use Notion For:**
- Project tracking
- Meeting notes
- Company wiki
- CRM (for now)
- Product roadmap

**Setup:**
- Join workspace
- Bookmark key pages
- Learn basic formatting

**Templates:** Use existing templates, don't create from scratch

---

### 4. Zoom - Video Calls

**Setup:**
- Install desktop app
- Test audio/video before first call
- Set virtual background (optional)

**Meeting Etiquette:**
- Mute when not speaking
- Video on for important meetings
- Use chat for questions
- Record if needed (ask permission first)

## Role-Specific Tools

### Product & Engineering

**GitHub:**
- Version control for code
- Issue tracking
- Pull requests and code review
- Access: Request from CTO

**Power BI Desktop:**
- Build and test templates
- Free download from Microsoft
- Training: Internal Power BI course

**Azure Portal:**
- Cloud infrastructure management
- Access levels: Dev, Staging, Prod
- Training: Azure fundamentals (online)

**Figma:**
- Design collaboration
- View-only for most, edit for designers/PMs
- Comment on designs

---

### Sales & Customer Success

**CRM (Notion):**
- Track leads and pipeline
- Log customer interactions
- Update deal status

**Loom:**
- Record async video messages
- Send personalized demos
- Training videos for customers

**Intercom (Future):**
- Customer messaging
- Live chat support
- Help center

---

### Finance & Operations

**QuickBooks / Xero:**
- Accounting software
- Access based on role
- Training: Finance manager

**Stripe:**
- Payment processing
- Dashboard for transactions
- Access: Finance only

---

## Security Tools

**Password Manager (1Password/LastPass):**
- Store all passwords securely
- Never reuse passwords
- Enable 2FA everywhere possible

**VPN (if remote):**
- Required for accessing prod database
- Access: Request from IT/CTO

## Getting Help

**Can't access a tool?**
- Slack #it-support or ask your manager
- Most access requests resolved <24 hours

**Tool not working?**
- Check status page (if available)
- Restart app
- Ask in Slack #questions

**Want to try a new tool?**
- Propose to your manager
- Business case required for paid tools
