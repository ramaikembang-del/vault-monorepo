# Core Skills Training (Employees)
**Foundational Skills for All Team Members**

**Owner:** HR + Operations  
**Status:** PLANNED - Year 1+

---

## Core Skills Overview

**Why Core Skills?**
- Baseline competency (everyone speaks same language)
- Cross-functional understanding (engineers understand sales, sales understand tech)
- Career growth (foundation for advancement)

**Who:** All employees (mandatory in first 90 days)

---

## Skill Categories

### 1. Communication Skills

**Written Communication:**
- Clear, concise emails (no jargon, action-oriented)
- Status updates (what/why/when format)
- Documentation (see [[biz/departments/operations/documentation/documentation-standards|Documentation Standards]])

**Verbal Communication:**
- Active listening (repeat back to confirm)
- Asking questions (clarify vs assume)
- Giving feedback (constructive, specific)

**Training Format:** 2-hour workshop + practice

---

### 2. Collaboration Tools

**Slack:**
- Channel etiquette (don't spam @here)
- Threaded conversations (keep organized)
- Status updates (set "Do Not Disturb" when focusing)

**Notion:**
- Find documents (search, navigation)
- Create pages (formatting, linking)
- Collaborate (comments, @mentions)

**Google Workspace:**
- Docs/Sheets/Slides basics
- Sharing permissions (view vs edit)
- Version control (named versions)

**Training Format:** 1-hour guided tour + hands-on

---

### 3. Time Management

**Prioritization:**
- Urgent vs Important (Eisenhower Matrix)
- Say "no" gracefully (when capacity is full)
- Batch similar tasks (context switching is expensive)

**Focus:**
- Deep work blocks (2-hour uninterrupted time)
- Minimize distractions (close Slack, silence phone)
- Pomodoro technique (25-min sprints)

**Training Format:** 1-hour workshop + personal planning session

---

### 4. Problem-Solving

**Framework:**
1. Define problem (what exactly is wrong?)
2. Root cause analysis (5 Whys)
3. Generate solutions (brainstorm 3+ options)
4. Evaluate tradeoffs (pros/cons)
5. Decide and execute

**Training Format:** Case study exercise (real company problem)

---

### 5. Data Literacy

**Basics:**
- Read charts (line, bar, pie)
- Interpret percentages (e.g., "Growth of 20%" means what?)
- Correlation vs causation (A and B move together â‰  A causes B)

**Application:**
- Dashboard interpretation (what do metrics mean?)
- A/B test results (statistical significance)
- Business metrics (MRR, CAC, LTV)

**Training Format:** 2-hour workshop with real data

---

### 6. Customer Empathy

**Understanding Customers:**
- Who they are (Phase 2-3 F&B owners)
- What problems they face (manual tracking, no insights)
- How we help (automated analytics)

**Exercises:**
- Read 10 customer support tickets
- Listen to 3 customer calls
- Use our product as a customer would

**Training Format:** Half-day immersion

---

### 7. Product Knowledge

**All Products:**
- Template (Tier 1/2/3)
- Support (R1/R2/R3)
- SaaS (Basic/Pro/Enterprise)

**Hands-On:**
- Install Template yourself
- Use SaaS dashboard (demo account)
- Navigate all features

**Quiz:** 30 questions, must score 80%+

**Training Format:** Self-paced (2-3 hours) + quiz

---

## Training Schedule

**Week 1 (Orientation):**
- Communication skills
- Collaboration tools
- Product knowledge (basics)

**Week 2-4 (Role-Specific  Training):**
- Deep-dive on role (Engineering, CS, Marketing)
- Shadow teammates
- First projects

**Week 5-8 (Advanced Core Skills):**
- Time management
- Problem-solving
- Data literacy
- Customer empathy

**Day 90 (Assessment):**
- Skills quiz (all 7 categories)
- Manager evaluation
- Certification: "Clario Core Skills Certified"

---

## Refresher Training

**Annual Refresher:** Half-day workshop (all team)
- Review core principles
- Learn new tools (if added)
- Practice scenarios

**New skill modules** (as needed):
- Security awareness (annual)
- Leadership skills (for managers)
- Advanced data analysis (for analysts)

---

## Success Metrics

**Completion:**
- 100% of employees complete within 90 days
- 90%+ pass certification quiz (first attempt)

**Application:**
- Improved communication (fewer misunderstandings)
- Faster onboarding (30-day vs 60-day productivity)
- Higher engagement scores
