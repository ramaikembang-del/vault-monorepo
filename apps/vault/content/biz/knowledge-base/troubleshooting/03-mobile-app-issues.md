## Mobile Browser Issues

### Dashboard Not Loading on Mobile

**Solutions:**
1. **Use Chrome/Safari:** Default browsers work best
2. **Clear browser app cache:** Settings → Apps → Browser → Clear Cache
3. **Enable JavaScript:** Should be on by default
4. **Disable data saver:** Can block dashboard elements

---

### Layout Broken on Mobile

**Symptoms:**
- Text overlapping
- Buttons cut off
- Charts too small

**Solutions:**
1. **Rotate to landscape:** Some charts better in landscape
2. **Update browser:** Old mobile browsers have layout bugs
3. **Zoom out:** Pinch to zoom out if elements overlapping

**Workaround:** Request desktop site (Browser menu → "Desktop site")

---

### Can't Tap Buttons

**Solutions:**
1. **Zoom in:** Button might be too small
2. **Remove screen protector:** Some protectors reduce sensitivity
3. **Clean screen:** Fingerprints can interfere
4. **Try different finger/stylus**
