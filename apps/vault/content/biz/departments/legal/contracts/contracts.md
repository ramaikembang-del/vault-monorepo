# Legal Contracts

**Purpose:** Repository for contract templates and executed agreements.

---

## Contract Categories

This folder will contain templates and executed contracts for:

### Customer Contracts
- **Terms of Service (ToS)** - All products
- **SaaS Subscription Agreement** - Cloud products
- **Professional Services Agreement** - Template setup/support
- **Data Processing Agreement (DPA)** - GDPR/data privacy compliance

### Vendor/Supplier Contracts
- Software licensing agreements
- Hosting/infrastructure agreements (AWS, Azure, etc.)
- Payment processor agreements (Midtrans, etc.)
- Marketing/advertising agreements

### Partnership Contracts
- POS vendor partnership agreements
- Accounting software integration agreements
- Referral/affiliate agreements
- Revenue sharing agreements

### Employment & Contractors
- Employment agreements
- Contractor/freelancer agreements
- NDA (Non-Disclosure Agreement)
- IP assignment agreements

**IMPORTANT:**
- [x] Commit templates to git
- [x] DO NOT commit executed/signed contracts (contain PII)
- Use separate secure storage for executed contracts

## Legal Review Process

1. **Draft:** Use template or create from scratch
2. **Review:** Legal counsel review (if budget allows) OR online legal service
3. **Approval:** Founder sign-off
4. **Execution:** Both parties sign
5. **Storage:** Secure storage (Google Drive + backup)
6. **Tracking:** Log in contract register (spreadsheet)
