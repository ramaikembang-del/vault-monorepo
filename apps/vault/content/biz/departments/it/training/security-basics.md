# Security Basics for Team Members  
**Essential Security Hygiene**

**Owner:** IT + HR  
**Status:** ACTIVE

---

## Password Security

**Requirements:**
- Minimum 12 characters
- Mix of upper, lower, numbers, symbols
- Unique per account (no reuse)

**Use Password Manager:** 1Password or Bitwarden (company-provided)

**Never:**
- Share passwords (even with teammates)
- Write passwords on paper/notes
- Use "password123" or similar

---

## 2FA (Two-Factor Authentication)

**Required for:**
- Email (Gmail)
- GitHub
- Cloud services (AWS, Supabase)

**Recommended for:** All accounts that support it

**Method:** Authenticator app (Google Authenticator, Authy)

---

## Device Security

**Laptop/Desktop:**
- Full-disk encryption enabled
- Screen lock after 5 min idle
- Auto-update OS & software

**Mobile:**
- PIN/fingerprint lock
- Find My Phone enabled
- Work email in separate profile (if possible)

---

## Phishing Awareness

**Red Flags:**
- Urgent requests ("Act now or account locked!")
- Suspicious sender (check email address carefully)
- Requests for passwords or personal info

**If Unsure:** Forward to security@clario.id, don't click links

---

## Data Handling

**Customer Data:**
- Access only on need-to-know basis
- Never share outside company
- Delete from local device after use

**Sensitive Info:**
- Use encrypted channels (not plain email)
- Store in secure locations (1Password, not Notion)

---

## Incident Reporting

**If Security Issue:**
1. Report immediately to security@clario.id
2. Don't try to fix yourself
3. Document what happened

**Examples:** Lost laptop, clicked phishing link, password compromised
