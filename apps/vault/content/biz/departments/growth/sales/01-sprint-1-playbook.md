# Sprint 1 Sales Playbook

**Founder-Led Sales | WhatsApp-First Approach**

---

## Power BI Template Sales Process

### 1. Lead Sources (Week 1-4)

**Warm Leads (Priority):**
- Bandung research participants (10+ contacts)
- Personal network referrals
- F&B owner WhatsApp groups

**Cold Outreach:**
- Instagram followers (engaged with content)
- LinkedIn connections (F&B industry)

---

### 2. Initial Contact Script (WhatsApp)

**Warm Lead Opening:**
```
Halo [Name]!

Masih inget waktu kita ngobrol soal [pain point mereka]?

Aku udah buat solusinya: Dashboard Power BI lengkap khusus F&B, tinggal pakai. Harga Rp 2-3M aja (hemat 90% vs bikin custom).

Mau aku kirimin demo video 5 menit? Dijamin mata langsung terbuka lihat data bisnis kamu
```

**Cold Lead Opening:**
```
Halo!

Liat dari [Instagram/LinkedIn], bisnis F&B nya udah jalan [X] outlet ya?

Udah pake dashboard buat tracking sales, COGS, sama staff performance?

Atau masih pakai Excel manual?

Aku punya solusi dashboard Power BI siap pakai, Rp 2-3M. Mau liat demo nya?
```

---

### 3. Demo Process

**Step 1: Send Demo Video (5-7 min)**
- Overview of 3 tiers
- Key dashboard walkthroughs
- Price reveal and value comparison

**Step 2: Schedule Call (15-30 min)**

**Demo Call Structure:**
1. **Discovery (5 min)**
 - "Cerita dong, sekarang tracking bisnis pakai apa?"
 - "Paling pusing di data yang mana?" (sales/COGS/staff/inventory)
 - "Pernah coba pakai dashboard? Kendalanya apa?"

2. **Live Demo (10-15 min)**
 - Show dashboards matching their pain points
 - Use sample F&B data that looks like theirs
 - "Bayangin kalau ini data [bisnis mereka]..."

3. **Pricing & Next Steps (5-10 min)**
 - Present 3 tiers
 - Recommend based on their tech capability
 - "Kalau aku lihat, [Tier X] cocok karena..."

4. **Handle Objections** (see below)

---

### 4. Objection Handling

#### "Mahal, ga sanggup Rp 2-3M"

**Response:**
```
Aku paham. Tapi coba kita hitung:
- Bikin custom development: Rp 20-50M + bulanan maintenance
- Hire data analyst: Rp 5-10M/bulan
- Template ini: Rp 2-3M ONE TIME, pake selamanya

Kalau mau lebih murah lagi, ada Tier 1 (DIY) Rp 2M. Atau coba Web Dashboard Lite kami, gratis. Mau link nya?
```

#### "Ribet ga sih setupnya? Aku ga ngerti Power BI"

**Response Tier 1 (DIY):**
```
Makanya ada panduan lengkap + video tutorial. Setup 2-4 jam doang.

Tapi kalau mau dibantu, ambil Tier 2 (Assisted). Kami setup in bareng sampe jalan. Rp 5-7M.

Atau mau yang fully managed? Tier 3, kita yang urus semua. Rp 10-15M + Rp 300-500K/bulan.
```

#### "Nanti kalau ada bug atau perlu update gimana?"

**Response:**
```
Nah makanya ada Template Support! Rp 75K/bulan:
- Dapat update template terbaru (kita terus tambahin fitur)
- WhatsApp support <4 jam
- Tips & best practices bulanan

First month FREE kalau beli sekarang
```

#### "Data aku di [software X], compatible ga?"

**Response:**
```
Power BI bisa connect ke hampir semua software (Excel, SQL, APIs).

Kalau software kamu ada export Excel/CSV, pasti bisa.

Mau kita cek dulu compatibilitynya? Kirim screenshot software nya aja.
```

#### "Mau mikir dulu / diskusi sama partner"

**Response:**
```
Oke, ga masalah! Mau aku kirimin proposal lengkap buat didiskusiin?

Ada perbandingan 3 tier, sample dashboards, sama pricing breakdown.

Oh iya, kalau ambil keputusan dalam [X hari], ada early bird discount [Y%]. Soalnya ini produk baru ```

---

### 5. Closing Techniques

#### Assumptive Close
```
"Oke kalau gitu, Tier [X] ya yang cocok.

Nanti aku kirimin invoice + detail rekening. Pas udah transfer, langsung kita schedule onboarding call buat setup. Deal?"
```

#### Urgency Close (Limited Time)
```
"Btw, ini launch offer doang. Bulan depan harga naik [Y%].

Kalau ambil keputusan minggu ini, dapet harga special + first month support FREE."
```

#### Alternative Close
```
"Kalau budget lagi tipis, mending ambil Tier 1 (DIY) Rp 2M dulu.

Nanti kalau butuh bantuan, bisa upgrade ke Tier 2/3. Gimana?"
```

---

### 6. Upsell: Template Support

**Timing:** During onboarding call OR 1 week after purchase

**Script:**
```
Gimana, udah coba template nya? Dashboard mana yang paling berguna?

Oh iya, tau ga kita ada Template Support? Rp 75K/bulan:
- Update terbaru (kita terus develop fitur baru)
- WhatsApp support kalau ada kendala
- Monthly tips analytics F&B

Karena udah beli template, first month FREE. Mau coba?
```

**Value Reinforcement:**
```
Jadi misal ada fitur baru atau improvement, otomatis dapet update.

Plus kalau ada error atau mau nanya best practice, tinggal WA aja. Reply <4 jam.

Worth it kan Rp 75K/bulan dibanding hire data analyst Rp 5-10jt?
```

---

### 7. Pipeline Management

**CRM:** Google Sheets (Sprint 1 bootstrap)

**Lead Stages:**
1. **New Lead** - Just made contact
2. **Demo Sent** - Sent video, waiting for response
3. **Call Scheduled** - Demo call booked
4. **Negotiation** - Discussing price/tier
5. **Closed Won** - Purchased
6. **Closed Lost** - Not interested / no response

**Follow-up Cadence:**
- Demo sent → Follow up in 24-48 hours
- Call done → Send proposal within 24 hours
- Proposal sent → Follow up in 2-3 days
- No response → Final follow-up after 7 days

## Web Dashboard Lite (Free) - Activation Focus

### Goal
Get 50+ users to download and USE (not just install)

### Distribution Channels
1. WhatsApp personal outreach
2. Instagram bio link
3. LinkedIn post
4. F&B community groups

### Activation Script (WhatsApp)
```
Halo!

Mau cobain Web Dashboard buat tracking bisnis F&B?

GRATIS, offline (ga perlu internet), simple banget:
1. Upload Excel omzet/expenses
2. Langsung dapet insights

Download di: [link]

Bingung cara pakenya? Chat aku aja! ```

### Engagement Strategy
- **Day 1:** Welcome message + quick start guide
- **Day 3:** "Udah coba upload data? Butuh bantuan?"
- **Day 7:** "Insight mana yang paling berguna?"
- **Day 14:** Feature tip: "Tau ga, kamu bisa [feature X]?"

**Success = Use 3x/week for 2+ weeks**

---

## Post-Sprint 1 Sales Strategy

**If Gate 1A passes (5+ template sales):**
- Scale outreach (hire SDR or VA)
- Build referral program
- Launch affiliate partnerships

**If Gate 1B passes (50+ Lite users):**
- Prepare paid tier upsell campaign
- Build conversion funnel tracking
- A/B test pricing (Rp 200K vs Rp 300K)
