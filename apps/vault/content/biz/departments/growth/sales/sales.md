## Phase 1 Sales Strategy

### Target Customers (ICP)

**Primary (Template Buyers):**
- Phase 2-3 F&B (Kecil-Menengah)
- 3-10 outlets or Rp 500M-2B revenue/year
- Has existing POS/accounting software
- Tech-savvy owner or has IT staff

**Secondary (Lite Users):**
- Phase 1 Mikro F&B
- Single outlet, manual Excel tracking
- Owner-operator, price sensitive
- Analytics-curious but commitment-averse

## Sales Funnel (Phase 1: Smart Solo)

### Template Sales Funnel
```
Awareness (WhatsApp/Instagram)
 → Inquiry (WhatsApp DM)
 → Demo (Video + call)
 → Purchase (Bank transfer)
 → Onboarding
 → [Upsell Support]
```

**Conversion Targets:**
- Inquiry → Demo: 60%
- Demo → Purchase: 40%
- Purchase → Support: 30%

### Lite User Funnel
```
Awareness (Social/referral)
 → Download (Free)
 → Activation (Upload data)
 → Engagement (Use 3x/week)
 → [Sprint 2: Paid conversion]
```
