## Power BI Template Onboarding

### Timeline
- **Hour 0:** Purchase confirmation
- **Hour 1-4:** Send welcome package
- **Hour 24-48:** Onboarding call scheduled
- **Day 2-7:** Hands-on setup assistance
- **Day 7:** Check-in + upsell support

---

### Step 1: Welcome Package (Immediate)

**WhatsApp Message:**
```
Terima kasih udah percaya sama PWBI F&B Growth!

Template Power BI kamu udah siap dipake. Ini yang akan aku kirim:

1. File template (.pbix)
2. Setup guide (PDF + video)
3. Sample data untuk testing
4. Akses support group (opsional)

Mau schedule onboarding call? Aku bantu setup sampai jalan.

Available:
- Besok jam [X-Y]
- Lusa jam [X-Y]

Pilih yang cocok ya!
```

**Email Backup:**
- Subject: "Template Power BI Kamu Siap! [Setup Guide Inside]"
- Attachments: Template file, PDF guide, sample data
- Call booking link (Calendly or manual)

---

### Step 2: Onboarding Call (30-60 min)

**Pre-Call Prep:**
- Review customer's business context (from sales notes)
- Prepare sample data matching their business type
- Test template on latest Power BI version

**Call Agenda:**

**1. Welcome & Context (5 min)**
```
"Halo [Name]! Gimana kabarnya?

Sebelum kita mulai, cerita dulu dong:
- Sekarang data nya ada dimana? (Excel, software, POS?)
- Paling pengen lihat insight apa duluan? (Sales? COGS? Staff?)
- Familiar ga sama Power BI? Atau first time?"
```

**2. Environment Setup (10-15 min)**
- Download Power BI Desktop (if needed)
- Open template file
- Review data source connections

**3. Data Connection (15-20 min)**

**If Excel/CSV:**
- Point Power BI to their data files
- Verify column mapping
- Refresh data → Show live dashboards

**If Software/POS:**
- Export data from their system
- Import to Power BI
- Set up refresh schedule (if tier 2/3)

**4. Dashboard Walkthrough (15-20 min)**
- **Sales Dashboard:** "Ini bisa lihat revenue harian, weekly, monthly. Lihat, ada tren naik turun..."
- **COGS Dashboard:** "Ini food cost percentage. Idealnya di bawah 30%. Kalau lewat, berarti ada yang bocor..."
- **Staff Dashboard:** "Bisa compare produktivitas staff. Siapa yang paling perform..."

**Focus on THEIR pain points from discovery call**

**5. Next Steps & Support Upsell (5-10 min)**
```
"Oke, sekarang template udah jalan. Next steps:
1. Coba explore sendiri dashboards nya
2. Upload data terbaru regularly (daily/weekly)
3. Share insights ke team kamu

Oh iya, tau ga ada Template Support? Rp 75K/bulan:
- Update template terbaru
- WhatsApp support <4 jam
- Monthly analytics tips

First month FREE. Mau coba?"
```

**6. Questions & Close**
- "Ada yang masih bingung?"
- "Aku follow up 3 hari lagi ya, cek udah lancar atau ada kendala"

---

## Churn Prevention

### Red Flags

**Template Buyers:**
- No onboarding call scheduled (24 hours)
- No dashboard usage (7 days)
- Multiple support tickets (frustrated)

**Support Subscribers:**
- No questions asked (30 days) = not seeing value
- Late payment / payment failed

**Lite Users:**
- No data upload (3 days)
- Single use only (no return)
- App uninstalled

---

### Intervention Playbook

**No Onboarding Call Scheduled:**
```
Halo [Name]!

Liat belum schedule onboarding call. Ada kendala?

Atau prefer setup sendiri? No problem, tapi kalau stuck langsung chat ya!

Aku available hari ini jam [X-Y] kalau mau quick call 15 menit
```

**No Usage (7 Days):**
```
Halo [Name], gimana template nya?

Liat dari [Tier 3 data / self-report] kayaknya belum dipake ya?

Ada yang menghambat? Setup stuck? Data ga cocok? Atau butuh bantuan lain?

Aku bantu sampai jalan kok! ```

**Multiple Support Tickets:**
```
Halo [Name],

Maaf ya kalau pengalaman nya kurang smooth

Mau schedule call buat resolve semua issues sekaligus?

Atau kalau memang produk nya ga cocok, aku bisa refund sebagian. Customer satisfaction yang utama!
```

**Lite User Churned:**
```
[If uninstalled within 7 days]

Halo! Liat kamu uninstall Lite. No worries!

Boleh tau kenapa? Biar kami bisa improve:
- Terlalu ribet?
- Fitur kurang?
- Ga sesuai kebutuhan?

Feedback kamu sangat berharga

[Incentivize with discount code for future if they give feedback]
```
