# Employee Onboarding Process  
**Welcome New Team Members**

**Owner:** Operations + HR  
**Status:** PLANNED - Launches when first hire (Year 1)

---

## Onboarding Timeline

### Pre-Day 1: Setup (Week Before)
- [ ] Send offer letter + contract
- [ ] Collect legal docs (ID, tax forms)
- [ ] Set up accounts (email, Slack, GitHub, Notion)
- [ ] Order equipment (laptop if needed)
- [ ] Assign onboarding buddy

---

### Day 1: Welcome
**Morning:**
- Welcome breakfast/video call
- Company overview (mission, values, products)
- Team introductions

**Afternoon:**
- IT setup (password manager, 2FA, VPN)
- Security training (required)
- First task assigned (small win)

---

### Week 1: Foundation
- [ ] Read team handbook
- [ ] Shadow team meetings (standup, retro)
- [ ] 1-on-1 with manager (set expectations)
- [ ] Access to tools (Notion docs, codebase)
- [ ] First PR or deliverable

---

### Week 2-4: Ramp Up
- [ ] Take on real projects (with buddy support)
- [ ] Customer interaction (if CS/Sales role)
- [ ] Feedback session (how's onboarding going?)

---

### Day 30: Check-In
**30-Day Review:**
- Manager + new hire discussion
- What's working? What's confusing?
- Adjust role/expectations if needed

---

### Day 90: Performance Review
**Topics:**
- Competency assessment (meeting expectations?)
- Goals for next 90 days
- Career growth discussion

---

## Role-Specific Onboarding

**Engineering:**
- Week 1: Set up development environment
- Week 2: Fix first bug (starter issue)
- Week 3: Ship first feature

**Customer Success:**
- Week 1: Shadow senior CS on customer calls
- Week 2: Handle tickets under supervision
- Week 3: Own customer accounts

**Marketing/Sales:**
- Week 1: Learn product (use it yourself)
- Week 2: Listen to sales calls
- Week 3: First demo or content piece

---

## Onboarding Checklist (Manager)

**Before Day 1:**
- [ ] Laptop + access ready
- [ ] Buddy assigned
- [ ] Welcome email sent

**Week 1:**
- [ ] Daily check-ins
- [ ] Intro to key stakeholders
- [ ] First small task completed

**Week 2-4:**
- [ ] Real project assigned
- [ ] Feedback collected
- [ ] Address blockers

**Day 30:**
- [ ] Formal review
- [ ] Adjust plan if needed

**Day 90:**
- [ ] Performance assessment
- [ ] Decide: Continue, extend probation, or part ways
